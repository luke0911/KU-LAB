# 교차 검증 결과 보고서

## 검증 방법
- 7개 데이터셋 step_analysis TSV 사용 (총 713 ground + 286 stair steps)
- Mann-Whitney U 검정 (비모수), IQR 겹침률, Effect Size (rank-biserial)
- 판정: p<0.01 AND IQR겹침<30% → ✅강함, p<0.05 → ⚠️약함, else → ❌실패

## Claim별 검증

### Claim 1: procRmsZ 절대 임계값 실패 → 부분적 맞음
- 통계적으로는 가장 강한 분리력 (p=2.2e-16)
- 하지만 절대 임계값 2.0 적용 시: 거칠게걸음 Prec=0.37, 살살걸음 F1=0.20
- 결론: 방향은 맞지만 절대값 임계값이 일반화 안 되는 것이 핵심 문제

### Claim 2: varYZRatio < 0.8 → ❌ 불일치
- COMBINED: p=0.087 (유의하지 않음), IQR겹침 77.2%
- 7개 중 5개 데이터셋에서 ❌실패
- varYZRatio < 0.8 임계값 적용 시 F1: 0.30~0.52 (낮음)
- 결론: 분석에서 주장한 "예외 없이 0.2~0.7로 떨어짐"은 과장

### Claim 3: stepVarH ground > stair → ✅ 대체로 맞음
- 7개 중 5개 데이터셋에서 ✅강함 또는 ⚠️약함
- 거칠게걸음 1.77x, sim_data 1.93x → 강한 방어 효과
- 예외: sample(0.99x, 분리 없음)
- 결론: 유효하나 보편적이지는 않음

### Claim 4: zPeakYRatio lower=stair → ⚠️ 약함
- 7개 중 1개만 ✅강함 (거칠게걸음)
- COMBINED: p=1.3e-03이지만 IQR겹침 65.9%
- 결론: 거칠게걸음에서만 유효, 일반화 어려움
